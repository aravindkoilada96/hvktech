# hvktechinnovations.in Next.js Rebuild

Production-ready Next.js (App Router) rebuild of `https://hvktechinnovationsin.godaddysites.com/` using TypeScript and Tailwind CSS.

## Stack

- Next.js 15 (latest stable generated by `create-next-app`)
- TypeScript
- Tailwind CSS
- Nodemailer (contact form email delivery)

## Contact Form Email Setup

The contact form posts to `POST /api/contact`.

Create `.env.local` in project root:

```env
SMTP_HOST=smtp.example.com
SMTP_PORT=587
SMTP_USER=your_smtp_user
SMTP_PASS=your_smtp_password
SMTP_FROM_EMAIL=no-reply@yourdomain.com
CONTACT_TO_EMAIL=you@yourdomain.com
```

Notes:

- For SSL SMTP providers, use `SMTP_PORT=465`.
- The API route validates required form fields and returns meaningful error messages.

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production Build

```bash
npm run build
npm run start
```

## Deploy

### Vercel (recommended)

1. Push this folder to a Git repository.
2. Import the repo in Vercel.
3. Add environment variables listed above.
4. Deploy.

### Any Node Host

1. Set environment variables.
2. Run `npm run build`.
3. Run `npm run start` with a process manager.
